# postbox

A tiny journal for paper letters and postcards.

Some letters arrive.

Some disappear.

All are remembered.

## Goal

Keep track of outgoing and incoming paper mail.

Record when a letter was sent, when it arrived (if it did), and how long the journey took.

## Planned

- Send and receive journal

- Transit time statistics

- Telegram bot

- PostgreSQL storage

## Tech stack

`Python` · `Telegram Bot API` · `SQLAlchemy` · `Alembic` · `PostgreSQL`
